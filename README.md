# WordPress Consignment Terms Plugin

## Description
Simple purchase terms agreement plugin for WooCommerce checkout that matches the consignment-manager system.

## Features
- Yellow warning-styled terms box
- 3 specific purchase terms:
  * Immediate Shipping (with insurance)
  * No Refunds on Handmade Items
  * 2-3 Week Delivery Time
- Checkbox: "I agree to these purchase terms" (required)
- Initials field: "Your initials (e.g., JD)" (required)
- Form validation before checkout
- Saves to order metadata
- Displays in admin panel and emails

## Installation
1. Upload consignment-terms.php to /wp-content/plugins/
2. Activate plugin in WordPress admin
3. Configure WooCommerce checkout settings

## Version
6.0 - Simple Classic Checkout Version
